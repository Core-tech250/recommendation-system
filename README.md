# Movie Recommendation System

## Overview
This project implements a content-based movie recommendation system using the TMDB dataset.
The system recommends similar movies based on movie metadata such as genres, cast, crew, and keywords.

## Dataset
- TMDB 5000 Movies Dataset
- TMDB 5000 Credits Dataset

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- CountVectorizer
- Cosine Similarity
- Jupyter Notebook

## Methodology
- Data cleaning and preprocessing
- Feature extraction from textual data
- Vectorization using NLP techniques
- Similarity calculation using cosine similarity

## Result
The system successfully recommends movies similar to a given movie input.

## Example
Input Movie: Avatar  
Recommended Movies: Interstellar, Guardians of the Galaxy, Star Wars
